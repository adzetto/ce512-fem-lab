MATLAB Compatibility Aliases#

The package also exposes the original classroom script names as documented Python wrappers with explicit arguments:

MATLAB name

Python compatibility wrapper

canti

femlabpy.canti()

flow

femlabpy.flow()

flowq4

femlabpy.flowq4() or femlabpy.flowq4(T1, X, G, C, P=None, dof=1)

flowt3

femlabpy.flowt3() or femlabpy.flowt3(T2, X, G, C, P=None, dof=1)

bar01, bar02, bar03

femlabpy.bar01(), femlabpy.bar02(), femlabpy.bar03()

square, hole

femlabpy.square(), femlabpy.hole()

elastic

femlabpy.elastic(...)

nlbar

femlabpy.nlbar(...)

plastps, plastpe

femlabpy.plastps(...), femlabpy.plastpe(...)

These wrappers are designed for help() / docstring discovery and for users coming directly from the original MATLAB notes. Internally they call the vectorized femlabpy kernels and solvers.
